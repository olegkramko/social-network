
# Twitter Clone

This is a simple web application for sharing short massages and pictures.

To login as Administrator use:

Username: admin
Password: admin

# Tech Stack
* Java
* Spring Framework
* PostgreSQL
* Maven
* Bootstrap

# Functionality
* Registration
* Login/Logout
* Send picture
* Send message
* Delete message
* Edit message
* Add tag to message
* Search
* Email change
* Email verification
* Google Recaptcha verification
* User profile
* Message list
* Follow/Unfollow user
* Edit user(only for user with admin permissions)
* User roles
* Passwords encryption
* Pagination(number of pages and elements)
